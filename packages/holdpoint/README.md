# holdpoint

> `npx holdpoint check` — enforce eval checkpoints on AI coding agents.

This is a thin package so you can run Holdpoint without the scoped name:

```sh
npx holdpoint init
npx holdpoint check
npx holdpoint validate
npx holdpoint update
```

All commands are provided by [`@holdpoint/cli`](https://www.npmjs.com/package/@holdpoint/cli). See the [Holdpoint docs](https://holdpoint.dev) for the full reference.
